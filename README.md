# SQL-Sale-Dashboard
The dashboard helps business users quickly understand sales performance and make data-driven decisions.
The Sales Performance Dashboard is an interactive Power BI dashboard developed to analyze and visualize sales data. It provides important business insights such as total sales, number of orders, average sales, highest sale, regional sales performance, product-wise sales distribution, and sales trends.

**Project Objectives**

**The main objectives of this project are:**

Analyze sales data efficiently.
Display key performance indicators (KPIs).
Compare sales across different regions.
Identify best-selling products.
Visualize sales trends.
Create an interactive dashboard using Power BI.

**Dashboard Features**
Interactive visuals
KPI Cards
Sales Comparison
Product Distribution
Regional Analysis
Simple User Interface
Beginner Friendly
